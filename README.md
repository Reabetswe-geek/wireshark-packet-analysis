#Wireshark Packet Analysis

## Project Overview
This project demonstrates  capturing and analyzing network traffic using wireshark.

## Tools Used

- Wireshark
- Ubuntu (WSL)
- ICMP Ping
- DNS Queries

## Tasks Performed

- Installed Wireshark on Ubuntu
- Captured live network traffic
- Filtered DNS, HTTP, and ICMP packets
- Analyzed packet structure
- Saved packet capture (.pcap)

## Example Filters Used
dns
http
icmp
ip.addr == 8.8.8.8

## Files Included
network-capture pcap

## Key Learning
This project demonstrates understanding of network protocols and packet-level analysis used in cybersecurity investigation
